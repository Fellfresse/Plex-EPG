# Fellfresse-Plex-EPG

## About
Provides a 6 Days XMLTV EPG for PLEX DACH TV service.

## Usage
Grab the XMLTV file from this link and paste it into your favorite IPTV client:
```
https://raw.githubusercontent.com/Fellfresse/Plex-EPG/master/epg-de.xml.gz
https://raw.githubusercontent.com/Fellfresse/Plex-EPG/master/epg-at.xml.gz
https://raw.githubusercontent.com/Fellfresse/Plex-EPG/master/epg-ch.xml.gz
```
